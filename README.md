# CT Brain Image Segmentation Project

## 👤 Name:

Nafisat Ozohu Abdulsalam

## 📂 Project:

Segmentation of Brain CT Images Using Image Processing Techniques

## 🫁 Organ / Body Part:

Brain (Computed Tomography - CT)

## 📦 Dataset Used:

CT Medical Imaging Colorized Dataset (Kaggle)

## 🧠 Model / Method:

Classical image segmentation using:

* Grayscale conversion
* Otsu's Thresholding
* Morphological Opening (Noise Removal)
* Segmentation Mask Visualization
* Overlay of Segmented Region on Original CT Image

## 📊 Results:
No Dice or IoU score because the dataset did not contain ground truth masks.

### Qualitative Results:

* Successfully generated binary segmentation masks from CT images.
* Reduced image noise using morphological operations.
* Visualized segmented regions through overlay on the original CT scans.
* Demonstrated the application of classical image segmentation techniques to medical CT images.

## 🤔 Challenges Faced:

* The dataset does not provide ground-truth segmentation masks, making quantitative evaluation (Dice Score and IoU) impossible.
---

This project demonstrates a basic medical image segmentation workflow using traditional computer vision techniques. It provides an understanding of image preprocessing, threshold-based segmentation, mask generation, and visualization, serving as a foundation for more advanced deep learning-based segmentation methods.
